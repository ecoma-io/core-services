# iam-domain

Domain layer for IAM service: aggregates, value objects and domain events.

This library implements the core aggregates described in the architecture ADR and reuses the shared domain primitives from `@ecoma-io/domain`.

# iam-domain

## Building

Run `npx nx build iam-domain` to build the library.

## Running unit tests

Run `npx nx test iam-domain` to execute the unit tests via [Jest](https://jestjs.io).
